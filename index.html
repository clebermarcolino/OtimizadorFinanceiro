Solver Interativo

Modelagem matemática e solução ótima do problema de minimização de custos de produção em duas fábricas.

01 — Contexto

O Problema

Fábrica Desktops/dia Notebooks/dia Netbooks/dia Custo/dia
Manaus (x) 8.000 1.000 2.000 R$ 150.000
Sul (y) 2.000 1.000 7.000 R$ 210.000
Demanda mínima ≥ 16.000 desktops ≥ 6.000 notebooks ≥ 28.000 netbooks
02 — Modelo Matemático

Formulação em Programação Linear

Variáveis de Decisão
x = dias de operação da fábrica de Manaus
y = dias de operação da fábrica do Sul
Função Objetivo — Minimizar custo total
min Z = 150.000x + 210.000y
Sujeito às Restrições
8.000x + 2.000y ≥ 16.000  →  4x + y ≥ 8   // desktops
1.000x + 1.000y ≥  6.000  →   x + y ≥ 6   // notebooks
2.000x + 7.000y ≥ 28.000  →  2x + 7y ≥ 28  // netbooks
x ≥ 0 ,  y ≥ 0
